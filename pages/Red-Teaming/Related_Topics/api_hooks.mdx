To evade security controls, we first have to know how they work, the techniques they implement, then bypass each of those controls.

### 1. API Hooks
In the context of the Windows API, hooking is the process of intercepting function calls or messages passed between components of an application or operating system. This can be used for a variety of purposes, such as monitoring system events, modifying application behavior, or enhancing security.
